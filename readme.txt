Create google maps directly from your page/post!

== Description ==

Psyco maps is a very simple plugin to use, it allows you to create google maps directly from posts or pages with one or more markers with personalized title.

== Installation ==

How to install the plugin

1. Upload 'psyco-maps.zip' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Insert the Api of Google maps in psyco maps menu

== Frequently Asked Questions ==

= Where I find google API =

Go to https://developers.google.com/maps/documentation/javascript/get-api-key and click on generate key

= How to create maps =

Go to posts/page; add marker and your custom setting, click on generate maps and copy shortcode and paste it in the content of your page/post
